Semestrální práce "Závod kluzáků"

Ovládání:
Kluzák - kurzorové šipky na klávesnici
Kamera - WASD (pohyb), QE (rotace), PgUp a PgDown (posun nahoru/dolů), myš.
L - přepínání světla.

Konfigurace:
V souboru "swoopers.cfg" - zde se dají nastavit parametry kluzáku jako rychlost, model (byť aplikace počítá s výchozím modelem) a vykreslování bounding boxu (volba collidables_draw).

Knihovny třetích stran:
GLM - http://glm.g-truc.net/ (implementuje totožné API, jaké používá GLSL)
